Important Notice! regarding RequestPolicy

Multiple versions of RequestPolicy are installed and enabled in your browser. This will lead to conflicts!

Therefore, please follow these steps:

  1. Make sure that only one version of RequestPolicy is installed and enabled. You can disable other versions in the Add-ons Manager.
  2. Close this tab and restart your browser.
  3. If this tab does not open again, you're done. Otherwise you need to re-check the Add-ons Manager.

Sorry for the inconveniences.

Note: this is a static page, so it won't update even if you've fixed the problem. Simply close this tab when you're finished.
